Bem-vindo ao Curso de Flask

Neste guia, vamos construir uma aplicação completa utilizando Python Flask e MySQL. Você aprenderá desde a configuração inicial do ambiente até a criação de um sistema de autenticação e CRUD (Create, Read, Update, Delete) de usuários.

O que vamos usar?

  • Python 3.8+
  • Flask (Framework Web)
  • MySQL (XAMPP/WAMP)
  • VS Code

Configuração do Ambiente

Criando o ambiente virtual e instalando as dependências.

1. Criar e Ativar a venv

Abra o terminal na pasta do seu projeto e execute:

Terminal (Windows) 
python -m venv venv
venv\Scripts\activate

Nota: Se estiver no Linux/Mac use `source venv/bin/activate`

2. Instalar Bibliotecas

Instale o Flask e o conector do MySQL:

Terminal 
pip install flask mysql-connector-python

Arquivo: app.py

app.py 
from flask import Flask, render_template

app = Flask(__name__)

@app.route('/')
def home():
    return render_template('index.html')

if __name__ == '__main__':
    app.run(debug=True)

Conexão com MySQL

Configurando o banco de dados (XAMPP).

Script de Teste de Conexão

Crie um arquivo test_db.py para validar se o MySQL está acessível.

test_db.py 
import mysql.connector

try:
    mydb = mysql.connector.connect(
        host="localhost",
        user="root",
        password="", # Senha vazia é padrão do XAMPP
        # database="flask_crud" # Descomente depois de criar o banco
    )
    print("Conexão realizada com sucesso!")
except Exception as e:
    print(f"Erro ao conectar: {e}")

Automação Completa

Configuração de ambiente e banco em um único passo.

1. Arquivos SQL (Migrations)

Agora organizamos nosso SQL na pasta sql/. O arquivo 01_auth.sql contém a criação da tabela de usuários.

sql/01_auth.sql 
-- Criação do Banco
CREATE DATABASE IF NOT EXISTS flask_crud;
USE flask_crud;

-- Criação da Tabela de Usuários
CREATE TABLE IF NOT EXISTS users (
    id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(100) NOT NULL,
    email VARCHAR(100) NOT NULL UNIQUE,
    phone VARCHAR(20),
    password VARCHAR(255) NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- Admin Padrão
INSERT IGNORE INTO users (name, email, password) 
VALUES ('Admin', 'admin@sistema.com', '123456');
