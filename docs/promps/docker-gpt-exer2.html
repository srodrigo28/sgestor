📦 O que é Docker?

Docker é uma plataforma que permite rodar aplicações em containers, isolando dependências e versões.

✔ Resolve o problema: "na minha máquina funciona"

🔥 Por que é tão hypado?

  • Padroniza ambientes
  • Facilita deploy
  • Permite rodar qualquer projeto
  • Usado por grandes empresas

🔌 Portas

As portas conectam seu computador ao container.

React → 3000
PHP → 8000
Python → 5000

🖼️ Tipos de Imagens

  • node:20 → React
  • php:8.2-apache → PHP
  • python:3.11 → Python
